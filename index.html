Now, let's talk about types of data.

JavaScript has five primitive types of data:

  • String
  • Number
  • Boolean (true/false)
  • Null
  • Undefined

First, let's talk about Strings:

Strings can be any type of text, number, etc that you'd like, but only if it is surrounded by quotation marks ("sample" or 'sample'). You may use either double quote marks or single quote marks to make your data a string.
