For our final lesson, we will introduce you to loops. Loops can allow thousands of lines of code to be reduced down to only a few lines. There are a few types of loops, but for now, we will focus on the For Loop.

 

A basic For Loop is structured as follows:

for (where to start the loop; when to end the loop; size of increment)

 

Let's break this down a little. We first have to decide where we want to start the loop. To do this, we introduce a variable and assign it some value, let's say 0. Then, we need to decide when the loop should stop looping. That can be a number, and it can also be a variable. For example, your loop could start at 0 and end whenever the loop reaches the end of a list,and if you don't know how long that list is, then you may consider using the length of that list as your endpoint.

Now, we move onto the incrementor. Depending on the application of your loop, the incrementor can be a number or a variable. For now, let's leave the incrementor at 1, which is its most widely used value.

The typical incrementor constructor is "+=", which means you are adding and re-defining the variable to whatever the sum is. For example, var x = 1. If you do, x += 1, then x = 2. Then if you do x += 10, x will be 12! This is useful in loops to increment up by a set value on each iteration of the loop. Let's watch one run!
